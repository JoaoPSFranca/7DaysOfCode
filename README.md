# Desafio 7 dias de Código Alura

Este repositório foi criado para acompanhar o desafio de programação promovido pela Alura. 
Seriam 7 dias de desafio, onde em cada dia seriam apresentados 7 desafios de código para serem resolvidos em JavaScript. 
Os desafios se tratam de questões de lógica básica e utilização de alguns recursos do JavaScript. 

> Durante a data do desafio, os arquivos do dia 6 e 7 foram perdidos no processo, devido a isso, há apenas documentado os dias de 1 a 5.
